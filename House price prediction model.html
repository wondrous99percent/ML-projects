Scikit Learn Design¶

Primarily three type of objects

1.estimators - exmaple:imputer it has a fit method that fits the dtaset and calculates the parameters

2.transformers-transform methode takes input and returns output based on the learnings of fit

3.predicators-LinearRegression model.fit and predict are two common functions

Feature Scaling¶

It is used to scale all the features of the dataset into similar range

Primarily there are two types 1.Min-Max scaling(Normalization): Value-min/max-min class MinMaxScaler 2.standardisation: value-mean/standard deviation class StandardScaler
